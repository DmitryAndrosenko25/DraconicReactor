прописать инициализацию реактора при запуске программы

1. Проверить наличие файла reactorInfo.txt
2. если есть то
	- достать адрес реактора
	- проверить валидность адреса реактора    is reactor?
	=======================================================================
											r = requare("component")
											r.type("71bc94b4-e89e-430d-91e7-801f8511ef23") ----> String "draconic_reactor"
											if reactorType == "reactor" then
											  -- Это действительно реактор
											
											
											
											вернуть таблицу {"reactorAddres", isValidTxT(boolean), fluxIn, fluxOut}
	===============================================================
	если нет то 
	- запустить поиск реактора по списку component
	
3. После получения валидного адреса реактора прописать поведение под ВСЕ возможные состояния реактора 
	running, stopping, cold, warming_up, cooling
	
	- cold:
		- проверить fuelConversion > 30% - сообщение "ПЕРЕЗАРЯДИТЬ ТОПЛИВО РЕАКТОРА"
		- запустить алгоритм поиска адресов входного и выходного гейтов ДЛЯ ХОЛОДНОГО РЕАКТОРА
		- запустить алгоритм насыщения поля, сатурации и прогрева до 2000 градусов (перевести в статус warming_up и довести то возможности running)
	
	- warming_up:
		- запустить алгоритм поиска адресов входного и выходного гейтов ДЛЯ РЕАКТОРА В ПРОЦЕССЕ РАЗОГРЕВА
		- запустить алгоритм насыщения поля, сатурации и прогрева до 2000 градусов (довести то возможности running)
	
	- running:
		- запустить алгоритм поиска адресов входного и выходного гейтов ДЛЯ РЕАКТОРА В РАБОТЕ
		- поддерживать реактор в работе до достижения fuelConversion <= 90% (или сколько будет выставлено программой)
		
	-stopping:
		- запустить алгоритм поиска адресов входного и выходного гейтов ДЛЯ РЕАКТОРА ПРИ ОСТАНОВКЕ
		- поддерживать БЕЗОПАСНЫЙ заряд щита до (status cooling)
	
	
	
	- cooling:
		- запустить алгоритм поиска адресов входного и выходного гейтов ДЛЯ РЕАКТОРА ПРИ ОХЛАЖДЕНИИ
		- поддерживать БЕЗОПАСНЫЙ заряд щита до полного охлаждения (status cold)
		- после - ОКОНЧАНИЕ ПРОГРАМЫ
		
		
		
		
		
		
		
ПРАВКИ:
	- при температуре 13000 и fuelConversion > 40% - температура пригает на 0.1 градус вверх и даже на 0.2
